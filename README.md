### ğŸ Getting Started

#### Prerequisites

- Node.js (v18 or later)
- npm or yarn

#### Installation

1. Clone the repository
```bash
git clone https://github.com/salma247/portfolio.git
```

2. Install dependencies
```bash
cd portfolio
npm install
# or
yarn install
```

3. Run development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
